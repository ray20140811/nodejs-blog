## 為何使用 mysql 而不是 mongodb

- mysql 是企業內最常用的存儲工具, 一般都有專人運維

- mysql 也是社區內最常用的存儲工具, 有問題隨時可查

- 另: mysql 本身是一個複雜的數據庫軟件, 本課只講基本使用

## mysql 介紹

- web server 中最流行的關係型數據庫

- 官網可免費下載, 用於學習

- 輕量級, 易學易用
